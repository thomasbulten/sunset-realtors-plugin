# Store Locator

The Support Plugin for Sunset Realtors.

Built on the TAB IT Development plugin architecture with Vite.js asset bundling, WPBakery Page Builder integration, and automatic update checking.

## Features

- 📍 **Store Locator** - WPBakery component and `[store_locator]` shortcode
- 🚀 **Vite.js** - Modern asset bundling with hot module replacement
- 🎨 **Block Library** - WPBakery component registration
- 🔄 **Automatic Updates** - Built-in update checker for self-hosted plugins
- 🛠️ **Code Quality** - WordPress Coding Standards, ESLint, Stylelint, Prettier
- 🪝 **Git Hooks** - Husky pre-commit hooks for code quality
- 🌐 **i18n Ready** - Translation support built-in

## Requirements

- WordPress 6.8+
- PHP 8.2+
- Node.js >= 25
- npm >= 10
- WPBakery Page Builder (for the Store Locator component)

## Installation

1. Clone or download this repository
2. Navigate to your WordPress plugins directory:
   ```bash
   cd wp-content/plugins/
   ```
3. Place the `sunset-realtors-plugin` folder in your WordPress installation
4. Activate the plugin in the WordPress admin

## Setup

### 1. Install Dependencies

```bash
# Install PHP dependencies
composer install

# Install Node.js dependencies
npm install
```

### 2. Configure Vite.js

The plugin uses Vite.js for asset bundling. To use it:

1. **Development mode** (with hot module replacement):
   ```bash
   npm run dev
   ```

   Set the `VITE_WP_ORIGIN` environment variable to your WordPress site URL:
   ```bash
   export VITE_WP_ORIGIN=http://sandbox.local
   ```

2. **Build for production**:
   ```bash
   npm run build
   ```

3. **Watch mode** (rebuilds on file changes):
   ```bash
   npm run watch
   ```

### 3. Asset Structure

Assets are organized as follows:
- **Source files**: `assets/css/`, `assets/images/` and `assets/js/`
- **Built files**: `build/assets/` (generated by Vite)
- **Entry points**:
  - `assets/js/main.js` → Frontend JavaScript
  - `assets/js/admin.js` → Admin JavaScript

## Usage

### WPBakery

Add the **Store Locator** element from the WPBakery Page Builder component library.

### Shortcode

```
[store_locator]
```

## Versioning

This project follows [Semantic Versioning](https://semver.org/).

Version format: `MAJOR.MINOR.PATCH`
- **MAJOR**: Breaking changes
- **MINOR**: New features (backward compatible)
- **PATCH**: Bug fixes (backward compatible)

Update the version in:
- `sunset-realtors-plugin.php` (plugin header and `SUNSET_REALTORS_PLUGIN_VERSION` constant)
- `sunset-realtors-plugin.json`
- `package.json`
